Three-Steps Search Process

The searching process of a modern search engine can be summarized into three-steps, they are Crawling, Indexing, and The Search Process (Feng, n.d.).

 

Crawling

The search engine will uses software called bot, web crawler or spider to collect data such as URL, page title, internal links from websites on the web (Campbell, 2015). It will automatically collect necessary information from the web, and send these data back to the search engine database (Campbell, 2015).

 

Indexing

The search engine will then index the data collected the web crawlers by using keywords (Campbell, 2015). This process helps to organize these data to the site database (Campbell, 2015). The indexed database of the search engine enables users to get millions of relevant results within seconds.

 

The Search Process

Search engine will NOT search the data directly from the World Wide Web and will look from the indexed database instead. When the user entered their search query, the search engine server will up look the related information from its database and returns the search results to the user (Campbell, 2015).
