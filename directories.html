What is Directories?

We introduced the idea of directories in our introduction section. In this part, we will be
discussing the details of Directories.

Directory is a human-made index. It organizes links in a hierarchical order with different
categories and subcategories (Campbell, 2015). Users can drilling down different categories for specific
information (Campbell, 2015). The interaction between users, server, and directory sometimes rely on a
protocol called Lightweight Directory Access Protocol (LDAP) (Rouse, 2008).

Nowadays, man-made directory has been slowly replaced by automatic web crawlers and
algorithms for easier and faster organization of information. Nonetheless, some search
engines still uses manuel method to organize part of the search engine index (Brooks, 2008).
