What is Search Tool


 

 

Search Tool (Also known as Search Engine) offers searching services over the World-Wide-Web (The definition of search engine, n.d.). Users can type in keywords to formulate a search query, and use it define the general or specific information they are looking for (British Broadcasting Corporation, 2013). The information searched and listed by the Search Tool usually are mixture of webpages, images, videos, locations and other types of files (Techopedia, n.d.). Some search engine site, offers directories services, allowing the users to find the related information in categories.

 

 

A common misconception from the casual Internet users is that Search Tools will look up the information the user asked for directly from the Web. However, the actual search process differs much from this public assumption
